Pacchetto sito Alma Tellus

File inclusi:
- index.html: home con logo, Coming soon e menu hamburger
- soci.html: pagina richiesta di ammissione a socio
- grazie.html: pagina di conferma dopo invio
- style.css: grafica condivisa
- config.js: configurazione URL Web App Google

Istruzioni:
1. Carica tutti i file nella stessa cartella del sito.
2. Carica anche il logo ufficiale con nome esatto: logo.png
3. Apri config.js e sostituisci INCOLLA_QUI_URL_WEB_APP_GOOGLE con l'URL della Web App Google.
4. Pubblica su GitHub Pages o hosting Aruba.
5. Prova soci.html compilando una richiesta di test.

Campi modulo compatibili con Google Apps Script:
- nome
- cognome
- codice_fiscale
- data_nascita
- luogo_nascita
- indirizzo
- email
- telefono
- professione
- motivazione
- presa_visione_statuto
- privacy
- richiesta_adesione
- website, campo nascosto anti-spam
